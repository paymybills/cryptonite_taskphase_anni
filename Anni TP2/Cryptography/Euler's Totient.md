# Euler's Totient

```python
# Given values for primes p and q
p = 857504083339712752489993810777
q = 1029224947942998075080348647219

# Euler's totient function Ï•(N) = (p - 1) * (q - 1)
phi_N = (p - 1) * (q - 1)
print(phi_N)
```
